getting started


  • Neutral density (ND) filters help to reduce the amount of light entering into the camera. This creates the opportunity to use longer exposure times.
  • Different ND filters can be used for different effects. The ND Filters tab shows more information on when to use different ND filters.

perfecting the shot


  • Always use a tripod when doing long exposure photography. Any movement of the camera will create noise and distortion.
  • Compose your image before using the ND filter.
  • Set the lens to Manual Focus after attaching the ND filter and before pressing the shutter release. This will prevent the lens from changing the focus that you set before applying the ND filter.
  • Use a remote shutter release when using Bulb mode to prevent shaking of the camera and tripod.
